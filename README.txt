NOTE: these below compile & run instructions only apply when manual compiling with g++ in in the root directory of this project.
To compile:
make

To run:
./labinvaders

Controls:
R D F G to control player 1
Left CTRL to shoot for player 1
Arrow keys to control player 2
S to shoot for player 2
P to quit

-iPAC default keys are based on MAME
