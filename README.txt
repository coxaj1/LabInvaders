Supervised by Dr Marmorstein


NOTE: these below compile & run instructions only apply when manual compiling with g++ in in the root directory of this project.
To compile:
make

To run:
./labinvaders

Controls:
W A S D to control player 1
SPACE to shoot for player 1
Arrow keys to control player 2
RETURN/ENTER to shoot for player 2
P to quit
